## T16: Camera Controls & Input Wiring

Already completed alongside T12-T14:
- `main.cpp` calls `camera.update_from_input(input_state)` each frame
- Keyboard: arrows (pan), Q/E (zoom in/out), R/F (tilt up/down), A/D (rotate), 1/2 (mode switch)
- Mouse: left-drag for pan (2D) or orbit (3D), scroll for zoom/distance
- `Camera::set_mode()` switches between 2D ortho and 3D perspective, preserving visual scale
